# Mad House: How-To

Hands-on guides, tutorials, and references for everything the Mad House builds and operates.

Each topic folder contains Jupyter notebooks. Open them in VS Code, JupyterLab, or any notebook viewer.

## Topics

| Folder | What it covers |
|--------|---------------|
| `vibe-coding/` | Skills, agents, and prompting for AI-native development |
| `vps/` | VPS setup, maintenance, and Coolify deployments |
| `bots/` | Discord bot development and operational standards |
| `incubation/` | Project lifecycle from concept to shipped |
| `org-setup/` | GitHub org structure, repo standards, and routing |

## Structure

Each topic folder is further divided into sub-topics, each containing one or more focused `.ipynb` notebooks.

## Usage

Clone the repo and open any notebook:

```bash
git clone https://github.com/madebymadhouse/how-to.git
cd how-to
code .
```

Notebooks are written to be read linearly. Code cells are runnable but reading them is enough to follow the guide.
