# VPS

How to set up, deploy to, and maintain a production VPS.

| Notebook | What it covers |
|----------|---------------|
| `setup/vps-setup.ipynb` | First access, SSH keys, UFW, fail2ban, Docker, Coolify |
| `maintenance/vps-maintenance.ipynb` | Docker cleanup, firewall audits, log management, security checklist |
| `coolify/coolify-deployment.ipynb` | Deploying apps and services, env vars, domains, rollbacks |
